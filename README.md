# Tic-Tac-Toe 🎮

A fun, animated two-player Tic-Tac-Toe game built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, just open and play!

---

## Features

- Two-player gameplay (X vs O) in the browser
- Animated mark placement with a spring bounce effect
- Winning cells pulse with a color highlight
- Confetti burst on every win 🎉
- Random congratulation messages for the winner
- Persistent scoreboard tracking wins and draws across rounds
- Draw detection with a friendly message
- "Play again" keeps scores; "Reset scores" wipes everything clean

No build step or server required.

1. Download or clone all three files into the same folder.
2. Open `index.html` in any modern browser.
3. Start playing!

```
tictactoe/
├── index.html  ← open this
├── style.css

```

## How to Play

1. Player **X** always goes first.
2. Click any empty cell to place your mark.
3. The first player to get **three in a row** (horizontally, vertically, or diagonally) wins.
4. If all 9 cells fill up with no winner, it's a **draw**.
5. Click **Play again** to start a new round (scores are kept).
6. Click **Reset scores** to clear the scoreboard and start fresh.

## File Details

### `index.html`
The page structure. Contains:
- Game title
- Scoreboard (Player X | Draws | Player O)
- Turn indicator
- The 3×3 board grid (`#board`) — cells are generated by JavaScript
- Reset scores button
- Confetti overlay container
- Win/draw result overlay with play-again button

### `style.css`
All visual styling and animations:

| Selector / Animation | Purpose |
|---|---|
| `.cell` | Individual board cell, hover scale effect |
| `.cell.pop .mark` | Spring-in bounce when a mark is placed |
| `@keyframes winPulseX/O` | Colored pulse on winning cells |
| `.result-overlay` | Full-board overlay shown on win or draw |
| `@keyframes bounce` | Bouncing emoji in the result overlay |
| `.confetti-piece` | Individual confetti particle |
| `@keyframes fall` | Falling + rotating confetti animation |

### `game.js`
All game logic, organised into clear sections:

| Section | Responsibility |
|---|---|
| Constants | Win lines, confetti colors, congratulation messages |
| State | `board`, `currentPlayer`, `gameOver`, `scores` |
| `buildBoard()` | Creates 9 cell elements and attaches event listeners |
| `handleMove(index)` | Processes a player's move, updates DOM and state |
| `getWinLine()` | Checks all 8 win conditions, returns the winning trio or `null` |
| `highlightWinners(line)` | Staggers the winner-cell animation across the three cells |
| `updateTurnIndicator()` | Refreshes the "Turn: X/O" display |
| `updateScoreboard()` | Syncs score values to the DOM |
| `showResult(type, winner)` | Shows the overlay with win or draw content |
| `launchConfetti()` | Spawns 60 animated confetti particles |
| `resetRound()` | Clears the board for a new round, preserves scores |
| `resetScores()` | Zeroes scores and resets the round |

## Customisation Tips

**Change player colors** — edit these two lines in `style.css`:
```css
.cell.x-mark .mark { color: #7F77DD; } /* purple */
.cell.o-mark .mark { color: #1D9E75; } /* green  */
```

**Add more congratulation messages** — extend the array in `game.js`:
```js
const CONGRATS_MESSAGES = [
  'Congratulations, you crushed it!',
  // add more here
];
```

**Add an AI opponent** — replace the `handleMove` call for the second player with a function that picks an empty cell (random) or implements the minimax algorithm for an unbeatable AI.
